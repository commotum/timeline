# DeLiVoTr: Deep and light-weight voxel transformer for 3D object detection (2024)
Source: a29d32-2024.pdf

## Core reasons
- The paper adapts transformer-based feature extraction to LiDAR 3D detection by keeping voxel feature maps at the same scale while using attention to build receptive field, so the model can detect small objects without the semantic loss of downsampling.
- It explicitly groups voxels into regions and uses voxel intra- and inter-region transformers to fuse local and global features before rasterizing to BEV, showing a transformer architecture tailored to 3D voxel representations.
